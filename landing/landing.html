AutoPrompt for Copilot

Automate batch prompts on Microsoft Copilot, M365 Copilot, GitHub Copilot & Teams Copilot — run sample or custom prompts, one after another, hands-free.

Features

📋

Sample Mode

Pick from built-in prompt categories — General Knowledge, Coding, Science, Business, and more. Select individual prompts via checkboxes.

✏️

Custom Mode

Paste prompts directly (one per line) or upload a .txt, .csv, or .json file with your own prompts.

⏱️

Configurable Delay

Choose a wait time between prompts (3s – 30s) to let Copilot finish responding before the next one fires.

⏯️

Playback Controls

Pause, resume, or stop the batch at any time. Full control so you can intervene mid-run.

📊

Live Progress

Real-time progress bar, prompt counter, and a live countdown ticker while the batch is running.

🖥️

Page Overlay

A status overlay on the Copilot tab shows which prompt is active and how long until the next one.

How It Works

1

Open Copilot

Navigate to copilot.microsoft.com, m365.cloud.microsoft/chat, github.com/copilot, or Teams Copilot in any tab.

2

Click the Extension Icon

Open the AutoPrompt popup from the toolbar.

3

Choose Your Prompts

Select Sample or Custom mode and pick your prompts.

4

Set Delay & Start

Choose the delay between prompts and hit Start. Sit back and watch!

Supported File Formats

Upload your own prompt files in Custom Mode

.txt

One prompt per line — the simplest format.

.csv

Optional prompt, query, or question header column.

.json

Array of strings, objects with prompt keys, or { "prompts": [...] }.
