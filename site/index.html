Capabilities

Everything you need for
batch prompting

📋

Sample Mode

Built-in categories: General Knowledge, Coding & Tech, Science, Business. Select prompts individually with checkboxes.

✏️

Custom Mode

Paste prompts directly or upload .txt, .csv, or .json files. Total flexibility.

⏱️

Configurable Delay

Choose 3s, 5s, 10s, 15s, or 30s between prompts to let Copilot finish responding before the next one fires.

⏯️

Pause / Resume / Stop

Full playback controls. Intervene mid-batch at any time — pause to review, resume, or stop entirely.

📊

Live Progress

Real-time progress bar, prompt counter, and a live countdown ticker while the batch is running.

🖥️

Page Overlay

A floating overlay on the Copilot tab shows which prompt is active and how long until the next one.

Workflow

Up and running in 4 steps

1

Open Copilot

Go to copilot.microsoft.com or github.com/copilot in any Chrome tab.

2

Click the Icon

Open the AutoPrompt popup from your browser toolbar.

3

Pick Prompts

Select from samples or enter your own in custom mode.

4

Hit Start

Set your delay, press Start, and sit back.

Compatibility

Supported file formats

Upload your own prompt files in Custom Mode

Format Expected Structure
.txt One prompt per line
.csv Optional prompt / query / question header column; last column value used per row
.json Array of strings, array of objects with prompt/query/question keys, or { "prompts": [...] }

Installation

Coming Soon

In the meantime, you can install manually from the latest GitHub release.

Download v1.1.0 from GitHub
  1. Download and extract the release ZIP from GitHub
  2. Open Chrome and go to chrome://extensions
  3. Enable Developer mode (toggle in the top-right corner)
  4. Click Load unpacked and select the extracted folder
  5. The AutoPrompt for Copilot icon will appear in your toolbar
